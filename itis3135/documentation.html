Habitat Analysis

Different species require specific forest densities.

Use the following parameters to evaluate a location:

  • Tree canopy percentage
  • Proximity to water
  • Rodent population density
if (canopyDensity > 0.75) { return 'Suitable'; }
Identification Methods

Visual identification is often difficult in low light.

Acoustic monitoring is the primary method for tracking.

const hootPattern = [1, 0, 1, 1];

  • Eye color (Yellow vs Dark)
Nocturnal Equipment

Standard binoculars are insufficient for midnight tracking.

Infrared sensors provide the best results without disturbing the birds.

sensor.setMode('INFRARED');
Data Collection

All sightings must be logged with precise timestamps.

Ensure your coordinates are in decimal degrees.
